Git Resources and Practice
==========================

Here is a list of Git Resources you can use. We definitely recommend using them as resources when you get stuck or can't remember how to do something with Git. Reading through the first resource is very helpful. Bookmarking the Git cheat sheet is also helpful. The other 2 resources are just here for you to use if you find them helpful.

`Git Guide - Getting Started (Atlassian) <https://www.atlassian.com/git/tutorials/setting-up-a-repository>`_

`Git Cheat Sheet (Tower Git) <https://www.git-tower.com/learn/cheat-sheets/git>`_

`Git Immersion Tutorial <http://gitimmersion.com/>`_

`Common Git Problems - First Aid Git <http://firstaidgit.io/#/>`_
