Setup¶

Accounts¶

Make sure you can access accounts for each of the following sites, creating a new account if necessary.

GitHub: https://github.com/

Slack: You should have already received an invitation for the CoderGirl Slack Community via email. Please make sure you accept and setup your account.

Canvas: Canvas is used for assignments and other curriculum-related information. You can log in at learn.launchcode.org using the same email and password as your launchcode.org account. If you don’t know your password, you can reset it from the main LaunchCode site.

Install Applications¶

Install each of the following applications.

Google Chrome: https://www.google.com/chrome/browser/desktop/

Slack: https://slack.com/downloads/ (Install the desktop app, and optionally the mobile app for your phone)

Atom: https://atom.io/

Windows Users¶

Git for Windows: http://gitforwindows.org/
